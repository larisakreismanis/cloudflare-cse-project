# Cloudflare Customer Solutions Engineer Technical Project
## Overview
This project demos how Cloudflare services integrate with a simple origin application.

A Flask app runs on an Azure VM and returns request headers.
Cloudflare sits in front of the origin and provides routing, security, and authentication.

## Cloudflare Features Used
**Cloudflare DNS and Proxy**
**TLS encryption using Cloudflare Origin Certificates**
**Cloudflare Tunnel**
**Cloudflare Workers**
**Cloudflare Access (Zero Trust)**
**Cloudflare API**
